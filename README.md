# ONELAB Ventures Reconciliation System

This project is a reconciliation system for ONELAB Ventures. It provides tools for reconciling transactions and settlements, with supporting scripts and test files.


## Demo Links

You can find demos and more details at the following links:

- [Google Drive Demo](https://drive.google.com/file/d/1ZuL-zERzx1-HwapyP_C_N1ojVGnffV5H/view)

## Project Structure and File Contents

- **Claude chat.txt**: Contains chat logs or notes, possibly from interactions with Claude AI or other chat-based tools.
- **Final Prompt.txt**: Stores the final version of a prompt, likely used for AI or automation tasks.
- **Output.txt**: Output file, possibly containing results from running scripts or reconciliation processes.
- **Raw Prompts.txt**: Contains raw or initial prompts, likely for AI or automation workflows.
- **reconciliation.py**: Main Python script for reconciliation logic. Handles the core functionality for matching and reconciling transactions and settlements.
- **test_reconciliation.py**: Python test file for validating the logic in reconciliation.py. Contains test cases to ensure correctness and reliability.
- **transactions_settlements.py**: Script for handling transactions and settlements data. May include data loading, processing, or utility functions for the reconciliation process.
